  This project is a simplified Splitwise-style application designed to help users track shared expenses, split bills, and settle balances among friends, roommates, or groups. It allows users to add expenses, assign participants, and automatically calculate who owes whom.

Features:
-> User Accounts
    Users can sign up, log in, and manage their expense records.
-> Add Shared Expenses
    Create expenses, select the involved members, and input amounts.
-> Automatic Expense Splitting
    The system automatically calculates how much each member owes.
-> Group-Based Tracking
    Create groups (friends, trips, roommates) and manage expenses within each group.
-> Balance Overview
    View who owes whom and the total outstanding amounts.
-> Settle Up
    Mark payments as settled and update balances instantly.
